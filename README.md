# riot-api-nodejs

## V1.0.3

## description

A nodeJS library for the Riot Games Api

## install
```
  $ npm install riot-api-nodejs
```

## definitions
With *typings*:
```
  $ typings install --ambient riot-api-nodejs
```

With *TSD*:
```
  $ tsd install riot-api-nodejs
```
